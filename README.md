# system.book.hellohorizon
A sales call booking system for [HelloHorizon Ltd](https://hellohorizon.uk) that will allow clients to: 
- view availabilty
- choose a suitable time
- book a sales call

while remaing inside the HelloHorizon environment.

## Built with
- NextJS
- NextUI + TailwindCSS
- MongoDB
